# Podcast Writing Skill

A Claude Code skill for writing content meant to be heard rather than read.

## What This Skill Does

This skill provides guidance for crafting podcast scripts, episode outlines, voiceover copy, audio narration, radio segments, and any writing that will be spoken aloud to listeners.

The core principle: podcast writing is fundamentally different from writing for the page. Listeners get one pass. Every sentence must be understandable the first time it's heard.

## Who This Is For

This skill is useful if you:
- Write podcast episodes or scripts
- Create voiceover content or narration
- Produce audio content for radio, audiobooks, or spoken-word performances
- Need to convert written content into material that sounds natural when spoken
- Want to improve the clarity and flow of any audio-first content

## Key Principles

The skill covers 13 core principles for effective podcast writing:

1. Write for the ear, not the page
2. Use conversational language
3. Write for one person, not a crowd
4. Favor natural speech rhythms
5. Build in clear signposting
6. Lead with the interesting part
7. Use examples generously
8. One idea at a time
9. Avoid information dumps
10. Embrace strategic repetition
11. Write for the host's voice
12. Consider production elements
13. Always read aloud before finishing
