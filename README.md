# Claude Skills for Smart Crows

A collection of Claude Code skills for cybersecurity professionals and others looking to enhance their workflows.

Each skill is self-contained with its own directory, documentation, and installation instructions.

## Installation

Each skill includes an `INSTALLATION.md` file with step-by-step setup instructions for Windows, macOS, and Linux.

General steps:
1. Navigate to the skill directory
2. Follow the `INSTALLATION.md` file
3. Restart Claude Code
4. Verify the skill appears in `/skills`

## What is a Claude Code Skill?

A skill is a reusable methodology or prompt packaged for Claude Code. When installed, skills help Claude apply a consistent approach to your requests.

## Requirements

- Claude Code installed locally
- Access to your `~/.claude/` directory
- Basic terminal/command-line familiarity

## Contributing

Follow the structure of any existing skill:

```
skill-name/
├── README.md           (overview and principles)
├── SKILL.md            (the skill definition)
├── INSTALLATION.md     (setup instructions)
└── examples/           (optional)
```
